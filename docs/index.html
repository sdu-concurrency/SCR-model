Welcome to the Supply Chain Resilience Documentation

This comprehensive documentation site provides guides for all user roles of the Supply Chain Resilience Assessment Tool. Whether you're an administrator, session manager, or survey participant, you'll find the information you need to use the platform effectively.

📚 Documentation Overview

The Supply Chain Resilience Assessment Tool helps organizations identify vulnerabilities and assess capabilities across their supply chain. It works in four phases:

📋 Phase 1: Map the Supply Chain

Understand your supply chain structure and create assessment sessions for your organization.

🔍 Phase 2: Identify Vulnerabilities

7-step survey for participants to identify and rank vulnerabilities across 7 categories.

📊 Phase 3: Prioritize and Align

Cross-organizational analysis with charts and visualizations for session managers.

🎯 Phase 4: Prepare Action Plans

Use the aggregated data to create concrete action plans and improve resilience.

📖 About the Project

Project Background

This tool is developed as part of the Cybersikkerhed og Forretningskontinuitet (Cybersecurity and Business Continuity) project, funded by the Danish Industry Foundation (Industriens Fond) and carried out by researchers from the University of Southern Denmark (SDU) in partnership with the Royal Danish Military Academy (Forsvarsakademiet).

Project Name Cybersikkerhed og Forretningskontinuitet
Project Owner Syddansk Universitet (University of Southern Denmark)
Period 2023-2026
Funded by Industriens Fond
Project Website cyber-smv.dk
Project Leader Jan Stentoft, Professor, SDU - stentoft@sam.sdu.dk

The project integrates cybersecurity into supply chain resilience work for Danish SMEs. It builds directly on the earlier Supply Chain Resilience project (2021-2023).
